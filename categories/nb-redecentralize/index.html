    Windows [CONF] En el host: usuario. [INFO] IP o nombre del equipo. La IP puede consultarse mediante ipconfig en la consola (Inicio -> cmd.exe) (más info sobre ipconfig).     4. Identificación

    Cuando no se conoce el estado de un host, puede identificarse si la máquina está encendida mediante ‘ping ’ y conocer el sistema arrancado con la aplicación nmap (es posible que se requieran permisos de administración, sudo en el cliente).: nmap -O <nombreoipdelhost>' La GUI oficial Zenmap, permite hacerlo de forma gráfica, tanto desde GNU/Linux como desde Windows. En caso de disponer de más de un sistema operativo, si se utiliza GRUB como gestor de arranque, se puede utilizar el grub-reboot (en algunas distribuciones grub2-reboot) para reiniciar a uno en concreto. »
